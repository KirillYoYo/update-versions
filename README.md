## Project setup

```bash
$ npm install
```

## ENV
GITHUB_TOKEN	    Токен доступа GitHub для аутентификации API
GITHUB_USERNAME	    Ваше имя пользователя на GitHub
PORT	            Порт где запущен http серве

## START

```bash
$ npm run start
```

## SWAGGER

```bash

http://localhost:3001/api-docs/ or our port from env
```
